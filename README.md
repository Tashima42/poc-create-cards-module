# Create cards module

## Atenção - Leia antes
Node: v14.7.0    
Yarn: v1.22.4    
Docker: v19.03.12    

___    

1. Clone o repo: `git@github.com:Tashima42/poc-create-cards-module.git`
2. Crie uma pasta para o mongo: `mkdir docker-mongo && docker-mongo/mongodata`
3. Inicie o container: `docker-compose up -d`
